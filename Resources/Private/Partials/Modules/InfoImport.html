Минимальный набор полей для импортируема данных: "uid, pid".
Если значение столбца "uid" = "0" - произойдет создание новой записи.
Если значение столбца "uid" > "0" - произойдет обновление записи с указанным идентификатором (если указанной записи не существует в базе данных строка будет проигнорирована).
Если значение столбца содержит модификатор "[D]1" - произойдет удаление записи с указанным идентификатором.

Импорт связей:
- Добавить связь "[A]1,[A]2,[A]3" (attach);
- Удалить связь "[D]1,[D]2,[D]3" (detach);
- Удалить все связи "[T]" (detach all).

Значение столбца "pid" по умолчанию имеет значение "0" - исключение составляют модели с "use \Litovchenko\AirTable\Domain\Model\Traits\Pid;",
при которой записи таблицы могут размещяться в любой части дерева страниц в т.ч. в корне дерева сайта.
